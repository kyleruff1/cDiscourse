## Type contracts

```typescript
export type NodeLabelKind =
  | 'machine_observation'
  | 'user_allegation';

export type NodeLabelSource =
  | 'manual_tag'
  | 'auto_metadata'
  | 'lifecycle'
  | 'semantic_referee'
  | 'composition_mutation'
  | 'ai_classifier'
  | 'future_source';

export type NodeLabelSurface =
  | 'timeline_node'
  | 'selected_context'
  | 'inspect'
  | 'composer'
  | 'hidden';

export type NodeLabelDisposition =
  | 'rendered_now'
  | 'inspect_only'
  | 'composer_only'
  | 'hidden_sensitive'
  | 'future_source'
  | 'intentionally_silent';

export interface NodeLabelMark {
  id: string;
  rawKey: string;
  kind: NodeLabelKind;
  source: NodeLabelSource;
  label: string;
  shortLabel: string;
  description: string;
  defaultSurface: NodeLabelSurface;
  disposition: NodeLabelDisposition;
  priority: number;
  visibleByDefault: boolean;
  confidence?: 'low' | 'medium' | 'high';
}
```

`NodeLabelMark` → `AnnotationChipDescriptor` adapter (per UX-001.7 handoff contract):

```typescript
function toAnnotationChipDescriptor(mark: NodeLabelMark): AnnotationChipDescriptor {
  return {
    id: mark.id,
    label: mark.shortLabel,
    kind: mark.kind === 'machine_observation' ? 'semantic' : 'flag',
    iconHint: mark.kind === 'machine_observation' ? 'info' : 'warn',
    tooltip: mark.description,
    source: mark.kind === 'machine_observation' ? 'machine' : 'user',
    category: mark.source,
    ariaLabel: mark.kind === 'machine_observation'
      ? `Machine observation: ${mark.label}`
      : `User allegation: ${mark.label}`,
  };
}
```

## Display contract (binding caps)

**Timeline node:** max 1 Machine Observation + 1 User Allegation + overflow indicator `+N`.

Example rendering:
Observed · Source requested
Alleged · Needs source
+2

**Selected context:** max 3 Observations + 3 Allegations + overflow `+N`.

**Inspect popout:** full grouped detail using `InspectGroupHeader`:
Machine Observations

Source requested
Point stalled

User Allegations

Needs source
Scope challenge


**Composer:** composer-only Observations render via `RefereeBannerView.observationChips`; advisory and non-punitive; never mounted as node labels.

**Sources marked future_source / hidden_sensitive:** never render on Timeline node or selected context surfaces in v1. May appear in Inspect only if disposition is `inspect_only`.
