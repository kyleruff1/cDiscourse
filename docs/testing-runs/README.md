# Testing Runs

Manual argument counter-test run logs. Generated by the `argument-counter-runner` Claude Code skill.

Each file: `<YYYY-MM-DD>-<scenario-id>.md`

## Rules

- Aliases only — no real emails or passwords in log files.
- No secrets, JWTs, API keys, or auth tokens in log files.
- "No secrets exposed: yes" must appear in every log.

See `docs/argument-testing-skills.md` for how to create a run log.
