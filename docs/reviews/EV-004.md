## Summary

EV-004 ships a pure-TS visual-symmetry MAP that turns nine validation / semantic codes (`source_chain`, `evidence_debt`, `scope`, `definition`, `logic`, `causal`, `anti_amplification`, `synthesis_ready`, `max_depth_reached`) into typed `{ code, label, helperLine, iconHint, chipKind, edgeStyle, bandKind }` descriptors. Labels are read by-reference from `gameCopy.toPlainLanguage`; `synthesis_ready` and `max_depth_reached` additionally read `helperLine` + `iconHint` by-reference from `LIFECYCLE_UX_MAP` so drift is auto-tracked. The RULE-003 `IconHint` union is extended strictly additively (+4 values: `key_term`, `logic_chain`, `causal_arrow`, `crowd_slash`) with zero removals and zero reordering. The doctrine ¬ß3 `anti_amplification` phrase is pinned verbatim with two tests; `hot` is banned; R4 routing for `max_depth_reached` is encoded via `bandKind: 'resolved'` plus archived-or-resolved helperLine. Implementation matches the design 1:1.
